## 📌 About Us

**SKKU-CodeTail**은 성균관대학교 소프트웨어학과 기반 프로젝트 팀으로,  
프론트엔드 / 백엔드 / AI 기술을 결합하여 실전형 서비스를 개발합니다.

우리는 단순한 과제가 아니라  
👉 “실제로 사용 가능한 서비스”를 만드는 것을 목표로 합니다.
